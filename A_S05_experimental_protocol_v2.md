## 5.1 Experimental Concept

The fundamental observable distinguishing the entanglement-gravity hypothesis from standard physics is a **differential response** between two quantum systems that differ only in their entanglement structure. The experiment compares:

- **Branch A (entangled):** A programmable quantum system prepared in a high-entanglement state with nonzero S_ent — GHZ-like, cluster state, or high-entanglement-depth spin state
- **Branch B (reference):** The same system with entanglement suppressed or destroyed, driven to a minimally entangled state with S_ent ≈ 0

Both branches have the same qubit count, same control schedule, same gross energy budget, and same physical geometry. The only systematic difference is the entanglement structure. Any reproducible differential signal between branches is attributed to the entanglement source term.

**Why NISQ-scale rather than macroscopic atomic ensembles:**

The original experimental target (N = 10⁶ ⁸⁷Rb atoms in a GHZ state) was motivated by state-of-the-art atom interferometry sensitivity. However, the α_screen audit (see `alpha_screen_audit.md`) showed that the Gaussian spatial suppression factor for a macroscopic BEC gives α_screen ~ 10⁻⁴³ — 39 orders of magnitude below the heuristic estimate [10⁻⁴, 10⁻²] used in earlier signal predictions. Until α_screen is derived from first principles for macroscopic systems (OP3), the predicted signal for macroscopic ensembles is indeterminate.

NISQ systems with 50–100 qubits operate in a regime where: (a) entanglement can be directly characterized, (b) device geometry is programmable, (c) the system size is closer to where the Gaussian screening formula may be valid, and (d) deriving or bounding α_screen via Lindblad techniques is more tractable.

## 5.2 Platform Requirements

**Minimum platform specifications:**

| Parameter | Requirement | Current NISQ capability |
|-----------|------------|------------------------|
| Qubit count | 50–100 | 50–1000+ (multiple platforms) |
| Entanglement depth | ≥ 10 qubits verified | Achievable with current 2-qubit gates |
| Gate fidelity | ≥ 99% (2-qubit) | 99–99.5% (best platforms, 2025) |
| Coherence time | ≥ 100 μs | 100 μs–1 ms (superconducting); seconds (trapped ions) |
| State verification | Entanglement witnesses or subsystem tomography | Standard capability |
| Reset/reuse | Branch B preparation in same run | Available on all platforms |

**Candidate platforms:**
- Superconducting qubits (Google, IBM, Rigetti): high gate speed, programmable geometry, mature error characterization
- Trapped ions (IonQ, Quantinuum): long coherence times, high-fidelity 2-qubit gates, all-to-all connectivity
- Neutral atom arrays (QuEra, Atom Computing): programmable geometry, high qubit counts, Rydberg-mediated entanglement

## 5.3 State Preparation Protocol

**Branch A — Entangled state:**

Prepare an N-qubit state with verified entanglement depth k ≥ 10 via one of:
- GHZ state: (|0⟩^⊗N + |1⟩^⊗N)/√2 — maximum bipartite entanglement, fragile
- Cluster state: generated by CZ gates on a 2D lattice — robust to local errors, scalable
- One-axis twisted spin-squeezed state — achievable in neutral atom arrays at large N with near-GHZ entanglement depth

Verify entanglement using:
- Entanglement depth witness: W_k with expectation ⟨W_k⟩ < 0 certifying k-body entanglement
- Subsystem purity: Tr(ρ_A²) < 1/(entanglement depth) for half-system subsystem A
- Randomized benchmarking of entangling gate fidelity

**Branch B — Reference state:**

From the same initial state, apply:
- Controlled dephasing: apply random single-qubit Z rotations to each qubit independently, destroying off-diagonal coherences while preserving populations
- Thermal reset: couple to a warm bath for time τ_D ≫ τ_coherence

Verify that entanglement depth of Branch B is ≤ 1 (product state or separable).

## 5.4 The Predicted Signal — Honest Statement

The corrected differential acceleration formula (from OP4 resolution) for a bounded entropic source is:

$$\Delta a(R) = -\frac{\tilde{\kappa}_{\text{eff}} c^4}{8\pi k_B \ln 2} \cdot \frac{M_{\text{ent}}(<R)}{R^2} \tag{5.1}$$

where M_ent(<R) = 4π∫ρ_ent(r)r²dr is the enclosed entanglement entropy and R is the distance from the ensemble center to the test mass.

**The signal magnitude is currently indeterminate.** This is because:

1. α_screen is unknown for NISQ-scale systems (OP3 unresolved)
2. The coupling prefactor c⁴/(8πk_B ln 2) ~ 5 × 10⁶⁵ requires suppression of ~65 orders of magnitude to produce laboratory-scale signals, and the mechanism providing this suppression (Planck-scale renormalization, OP10) is unresolved

**What we can state:**
- The *structure* of the signal is correct: differential acceleration proportional to κ̃_eff, to M_ent, and falling as 1/R² outside the source
- The *falsification criterion* (Section 06) is valid regardless of the signal magnitude prediction
- The experiment will produce an upper bound on |κ̃_eff × M_ent / R²|, which constrains the product of the coupling and the effective entanglement source, providing a meaningful experimental result whether or not a signal is observed

## 5.5 Systematic Error Budget

The dominant systematics in the NISQ-based protocol and their mitigations:

| Systematic | Physical origin | Estimated magnitude | Mitigation |
|-----------|----------------|--------------------|-----------| 
| Imperfect state preparation | Gate errors, initialization fidelity | Equivalent to reduced entanglement depth | Embed entanglement verification in protocol; post-select on verified runs |
| Decoherence drift | Time-dependent noise spectra, laser power drift | Changes Branch A–B contrast over time | Randomize branch order; alternating A-B-A-B sequence; frequent calibration |
| Geometry-dependent coupling | Qubit layout, substrate, wiring | Source profile ρ_ent(x) depends on device | Explicit modeling of ρ_ent(x) for each architecture before prediction |
| Platform cross-talk | Qubit-qubit capacitive coupling, control line crosstalk | Spurious differential signal | Symmetrize control pulses; null reference circuits with same topology but no entanglement |
| Measurement back-action | Projective measurement perturbs state before signal integration | Differential decoherence between branches | Weak measurement protocols; use of ancilla readout |
| Ambiguous screening | α_screen unknown | Uncontrolled overall signal scale | State signal magnitude as indeterminate; focus on structural dependence on entanglement |

## 5.6 External Benchmarks

**Atom interferometry (Asenbaum et al. 2020):**  
Dual-species ⁸⁵Rb/⁸⁷Rb interferometer with T ~ 2 s free-fall time measured the Eötvös parameter:
$$\eta = [1.6 \pm 1.8 \text{ (stat)} \pm 3.4 \text{ (syst)}] \times 10^{-12}$$

Per-shot resolution: ~1.4 × 10⁻¹¹ g. This is the verified benchmark for differential acceleration sensitivity. Any entanglement-induced effect that behaves like a composition-dependent force must respect this bound, or couple in a way orthogonal to composition.

**MICROSCOPE (Touboul et al. 2022):**  
Eötvös parameter constrained at ~10⁻¹⁵ between Pt-Rh and Ti-Al-V test masses. A mapping from this to a κ̃ constraint has not been derived — both test masses are classical solids with S_ent ≈ 0, so the entanglement source term contributes equally (effectively zero) to both trajectories. **MICROSCOPE does not directly constrain κ̃ until this mapping is worked out.** This row has been removed from the experimental bounds table pending that derivation.

## 5.7 The Near-Term Goal

The goal of the NISQ experiment is **not** to claim the predicted effect is detectable with current hardware. It is to establish a regime in which:

- If a signal is observed: it can be plausibly attributed to entanglement structure (not systematics)
- If no signal is observed: the null result translates into a meaningful bound on κ̃ once α_screen and the source profile are better understood

This is scientifically valuable regardless of outcome. The experiment either confirms the framework or constrains it — both are progress.
